# footage-parser
Автоматизированный загрузчик вертикальных футажей для Reels/TikTok. Поиск по ключевым словам через Pexels API и сохранение на диск. Данный инструмент значительно упростит работу с созданием креативов для арбитража трафика, а также просто видеокриеторов.

Возможности скрипта:

- Сам ищет по словам: Пишешь что нужно, и он вытягивает это через апи.

- Только вертикалки: Скрипт сразу отсеивает горизонтальный мусор и оставляет только портретный формат под телефон.

- Качает пачками: Можно выставить число, и он загрузит тебе хоть 5, хоть 10 роликов за раз.

- Раскладывает по папкам: Он сам создает директории, чтобы у тебя на компе не было каши из файлов.
